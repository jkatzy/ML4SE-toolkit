This folder contains a notebook used to generate the input/target pairs for model evaluation. Additionally, it includes the plot for target complexity across cases.

The notebook creates the input-target pairs in three groups based on the amount of SATD in the file:
- one-smell: for cases in-smell and out-smell
- zero-smell: for base cases no-smell-comment and no-smell-method
- multiple-smell: for the multi-out-smell and preprocessed cases
